Отчет в УФССП по РСО-Алания по камеральным проверкам (Report_UFSSP)
===========================================

Скрипт применяется к выгрузке "**Результаты камеральных проверок для УФССП по РСО-Алания**" (в xls формате).
Выгружает по районам в CSV-формат.
Сохраняется отчет в файл "**Report_UFSSP_yymmddhhiiss.zip**" на рабочий стол в едином ZIP-архиве.

Установка:
----------

- Скопировать файл "**Report_UFSSP.xlsm**" в папку "**Мои документы**".
- Открываем файл в Microsoft Excel (желательно не ниже Office 2010).
- Зайти в меню > Файл > Параметры > Настройка ленты
- Создать вкладку
- Задать нужное имя вкладки "**Наименование организации**"
- Задать нужное имя группы "**Отчетность**"
- Выбрать команды: Макросы
- Выбрать макрос: Report_UFSSP.xmls!Report_UFSSP.Report_UFSSP
- Добавить, Переименовать в "**Отчет в УФССП по камералке**"
- ОК
  
Инструкция по запуску:
----------------------

- Открываем выгрузку "**Результаты камеральных проверок для УФССП по РСО-Алания**" (в xls формате).
- На ленте выбираем раздел "**Наименование организации**"
- Группа "**Отчетность**"
- Запускаем "**Отчет в УФССП по камералке**"
- Если файл выгрузке верен, пойдет процесс формирования отчета.
- По завершению процесса работы макроса выйдет сообщение с ссылкой на файл отчета.
